# Learn Bahasa Indonesia with HoloID

## Vocabulary Builder

- **[dictionary.csv](./dictionary.csv)**: Handcrafted Indonesian-English dictionary (WIP)
- **[vocab.csv](./vocab.csv)**: Most frequent Indonesian words appeared on HoloID streams
